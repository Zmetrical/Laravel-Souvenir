📋 Feature Modules (in build order)
Phase 1 — Design Builder ⭐ Hero Feature

Canvas workspace (Fabric.js on the frontend)
Drag-and-drop beads, charms, and accessories onto a base product
Change colors, resize, reposition elements
Save design as a JSON snapshot + exported image (stored via Laravel)
"Add to Order" button to carry design into checkout

Phase 2 — Order Submission & Tracking

Customer fills out order form (quantity, notes, contact info)
Design snapshot attached to the order automatically
Order confirmation page with a unique order code
Customer can check order status via their account

Phase 3 — Admin Dashboard

View all incoming orders (with design preview)
Update order status: Pending → Confirmed → In Progress → Ready → Completed
Basic sales summary (total orders, revenue)

Phase 4 — Product/Inventory Management

CRUD for products (keychains, bag charms, print types)
CRUD for charm/bead elements (what appears on the canvas)
Stock tracking per item
